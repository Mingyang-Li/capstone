# UoA capstone repo 

### 🛠️ Technologies used
| Purpose | Tools |
| :--- | :---- |
| ✅ Language | TypeScript |
| ✅ Authentication | Auth0 SDK |
| ✅ Routing | React-Router |
| ✅ State management | Apollo client |
| ✅ Styling - structure | MUI & Tailwind CSS |

### Run the app
```bash
# Install dependencies
yarn

# build
yarn build

# test
yarn test

# running
yarn start
```
